Projeto de Desenvolvimento Web com Bootstrap

Visão Geral

Este projeto foi desenvolvido durante uma das aulas de Desenvolvimento Web, utilizando a biblioteca Bootstrap para criar uma aplicação web responsiva e estilizada. Embora o projeto tenha sido concluído em junho de 2023, somente hoje decidi compartilhar o código no GitHub para possibilitar colaborações e fornecer uma referência para outros desenvolvedores.

Recursos e Funcionalidades

1. Barra de Navegação
Menu Responsivo: A barra de navegação oferece um menu responsivo que se adapta a diferentes dispositivos, garantindo uma experiência de usuário consistente em diferentes tamanhos de tela.
Botão de Entrar: Inclui um botão "Entrar" para direcionar os usuários a realizar ações específicas, como fazer login ou se cadastrar.

2. Seção Home
Mensagem Impactante: A seção inicial apresenta uma mensagem clara e impactante com título e descrição, destacando a proposta de valor do Finans.
Formulário de Cadastro: Inclui um formulário de cadastro simples com um campo de e-mail e um botão "Cadastre-se".
Compatibilidade com Dispositivos Móveis: A seção home é projetada para ser responsiva, garantindo uma boa experiência de usuário em dispositivos móveis.

3. Seção "Saiba"
Informações Detalhadas: Fornece informações detalhadas sobre o propósito do Finans, incentivando os usuários a explorar mais sobre a ferramenta.
Botão de Ação: Inclui um botão "Veja mais" que direciona os usuários para obter mais detalhes sobre a funcionalidade apresentada.

4. Seção "Juros"
Imagem Ilustrativa: Utiliza uma imagem ilustrativa para destacar o problema que o Finans ajuda a resolver.
Alertas Financeiros: Informa sobre a capacidade do Finans de enviar alertas sobre contas a pagar e a receber.

5. Seção "Recursos"
Três Recursos Principais: Destaca três recursos principais do Finans - "Fácil de usar", "Economize seu tempo!" e "Suporte Amigo".
Imagens Ilustrativas: Acompanha cada recurso com uma imagem ilustrativa correspondente.

6. Rodapé
Links de Navegação Rápida: Oferece links de navegação rápida no rodapé para "Home", "Recursos", "Benefícios" e "Preços".
Links de Mídias Sociais: Inclui ícones de mídias sociais com links para as contas do Finans, proporcionando aos usuários outras formas de se conectar.

7. Estilização e Personalização
Bootstrap CSS: Utiliza o framework Bootstrap para garantir um design moderno e responsivo.
Font Awesome: Integra ícones da Font Awesome para adicionar elementos visuais atrativos.
Estilo Customizado: Implementa um arquivo de estilo personalizado (estilo.css) para customizações específicas.

8. Compatibilidade com Dispositivos Móveis
Layout Responsivo: Todo o projeto é desenvolvido com um design responsivo, garantindo uma experiência de usuário consistente em dispositivos móveis.

10. Links Externos
Links para Download: Fornece botões de download para as versões Android e iOS do Finans.

12. Estrutura de Página Organizada
Uso Adequado de Seções: Organiza o conteúdo em diferentes seções, facilitando a navegação e compreensão do usuário.


Este README fornece uma visão geral das principais características e funcionalidades do projeto. Sinta-se à vontade para explorar e contribuir para tornar o Finans ainda mais robusto e eficiente!
